What it does

Everything you need,
on your phone

Mojorinth brings the full Modrinth experience to Android, designed ground-up for Mojo Launcher.

🧩

Browse Modrinth

Search and explore mods, shaders, resource packs, modpacks, data packs, and plugins — all from the Modrinth catalogue.

⬇️

One-tap Download

Install mods directly into your active Mojo Launcher instance. No file manager needed, no manual copying.

🛡️

LTW Renderer Safeguard

Automatically switches the renderer to LTW (GL4ES) when Sodium or Iris is detected — so your instance just works.

📦

Import & Export .mrpack

Import any .mrpack modpack from local storage, or export your current instance — mods, shaders, and resource packs included.

⚙️

Instance Manager

View, enable, disable, and delete mods, shaders, and resource packs in any instance. Filter, search, and sort at will.

🔍

Log Analyzer

Paste a crash log and get a plain-English explanation of what went wrong and how to fix it.

🎨

8 Themes

Modrinth, AMOLED, Light, Ocean, Sakura, Midnight, Sunrise, and Mint. Your launcher, your vibe.

📡

Offline Mode

No internet? No problem. Manage instances, import packs, and change settings fully offline.

Getting started

Set up in
5 steps

Mojorinth needs access to your Mojo Launcher instances folder. Here's exactly how to link it.

1

Install Mojo Launcher

Mojorinth is a companion app — you need Mojo Launcher installed first. Download it from their official source.

2

Install Mojorinth

Download the latest APK from the button above and install it. You may need to allow installs from unknown sources in your Android settings.

Im working on getting it google play :3

3

Tap "Link Folder" on the home screen

The instance picker card will guide you. Tap Link Folder to open Android's file picker.

4

Navigate to the instances folder

Tap the ☰ hamburger menu → MojoLauncher → instances. The exact path is Android/data/git.artdeell.mojo/instances.

5

Tap "Use this folder"

Android will ask for permission — grant it. Mojorinth will now see all your instances and you're good to go! 🎉

Questions

FAQ

Common questions about Mojorinth.

Do I need Mojo Launcher?

It is technically optional, HOWEVER Mojorinth is a companion app for Mojo Launcher. It manages files inside Mojo's instances folder. Without Mojo Launcher installed, most features won't work.

Why does it need folder access?

Android's storage system requires explicit permission to read and write files outside the app's own folder. Mojorinth uses Android's Storage Access Framework (SAF) to access only the Mojo Launcher instances folder — nothing else.

Why does Mojorinth switch my renderer to LTW?

Mods like Sodium and Iris use OpenGL calls that are incompatible with all other Mojo renderers. Mojorinth detects these mods and automatically switches to LTW (GL4ES) so the game actually runs. You can change the renderer manually in Mojo Launcher if needed. (Not recommended however)

Can I use mods not on Modrinth?

Yes! You can use the Instance Manager to view any files in your instance folders, regardless of where they came from. Downloading requires Modrinth, but managing works for all files. Reach out to me on Discord if you have any ideas for how to incorporate other mod sites.

Why does my new instance not show up in Mojo Launcher/Mojorinth?

Both apps need to be restarted to pick up new instances — Mojorinth shows restart buttons after every install to make this easy.

Is Mojorinth free?

Yes, completely free and open source. No ads, no tracking, no in-app purchases. Just a tool built for the community.
